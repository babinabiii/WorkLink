WorkLink

Inclusive Job Matching Mobile Application

Overview

WorkLink is a mobile application designed to connect inclusive job seekers such as Persons with Disabilities (PWD), Senior Citizens, and Students/Youth with verified employers. The platform allows users to discover job opportunities, apply for positions, access training resources, and interact with community support tools. Employers can post job opportunities and manage applicants, while administrators monitor the platform for safety and verification.

This project is developed as a final system project using React Native with Expo Router and Firebase.
